# Async Analyzer

A Python application for asynchronous data analysis.

## Project Structure

- `main.py` - Main entry point
- `analyzer/` - Core analyzer package
  - `loader.py` - Data loading functionality
  - `fetcher.py` - Asynchronous data fetching
  - `report.py` - Report generation
- `data/` - Data storage
  - `urls.json` - URL configuration
- `requirements.txt` - Python dependencies
